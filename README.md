## 🔌 Technical Skills & Domain Expertise

### 🧠 Firmware & Core Development
* **Low-Level Programming:** Bare-Metal Development, Register-level manipulation, Hardware Abstraction Layers (HAL).
* **Modern Embedded Systems:** Developing secure and memory-safe firmware using **Rust (Embedded Rust)** and C/C++.
* **Interrupts & Timers:** Configuring Hardware Interrupts, External Interrupts, and Timer-driven events.
* **Memory Optimization:** Writing resource-efficient firmware for memory-constrained microcontrollers.

* **Circuit Design & Diagnostics:** Troubleshooting physical circuits, analyzing signals, and power management.
